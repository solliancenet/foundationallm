# FoundationaLLM Roadmap

## Short-term (November 2023)

- User Portal
- Core API
- Starter deployment with Azure Container Apps (ACA)
- Starter deployment with Azure Kubernetes Service (AKS)
- Standard deployment with Azure Kubernetes Service (AKS)
- Data Source Hub with LLM-based resolver (LLM-based reasoning on data sources selection)
- Semantic Kernel release updates (aligning with the v1 release of the Semantic Kernel)
- Vectorization pipelines
- Azure AI Search (formerly Azure Cognitive Search) vector store
- Microsoft Presidio integration (in Gatekeeper API)

## Mid-term (December 2023)

- Whitelabeling support in User Portal
- Management API
- Public demo tenant
- Agent Hub with LLM-based resolver (LLM-based reasoning on agent selection)
- Scoring and analytics requests
- Other LLMs (cloud based)
- Hyperscale vectorization
- Security-aware data sources
- Structure parsers (in Gatekeeper API)

## Backlog (approved)

- Admin Portal
- CLI
- Jobs engine
- Hyperscale jobs
- Identity Server (Duende) integration
- Prompt Hub with LLM-based resolver (LLM-based reasoning on prompt selection)
- Other LLMs (on-premises)
- PostgreSQL vector store
- Schema overlays (in Gatekeeper API)
- Caching (in Gatekeeper API)
- Containerized LLM agents
- FoundationaLLM artifacts versioning
- Fast agent instantiation
- Integration of OpenTelemetry for tracing
